Variabler

Det finnes tre måter å deklarere/lage en variabel; var, let, const:

  • "var" er en utdatert metode for variabler, som kan overskrive og endres fritt. For webutvikling unngår vi å bruke "var"
  • "let" kan tilegnes en verdi som kan manipuleres og muteres fritt etter deklarering. Nyttig for tellere og verdier
  • "const" tilegnes en verdi som ikke kan overskrives. Den må tilegnes en verdi fra starten, men kan ikke få en ny verdi senere
var navnA;
let navnB;
const navnC = "Vi bruker likhetstegn (=) for å gi variabelen en verdi"

Datatyper

Det finnes tre måter å deklarere/lage en variabel; var, let, const:

  • String/Streng
  • Number/Nummer
  • BigInt
  • Boolean
  • Objekt
  • Symbol
  • Null
  • Undefined

Operasjoner

Referere

Verdien til en variabel kan hente og refere til andre variabler. Eksempel:

const navnA = 6
const navnB = navnA

Utregning

Vi kan legge sammen verdien til variabler ved hjelp and pluss (+) og (-). Eksempel:

const tallA = 6
const tallB = 3
const svar = tallA + tallB
