3. Variabler

Det finnes tre måter å deklarere/lage en variabel; var, let, const:

  • var er en utdatert metode for variabler, som kan overskrive og endres fritt. For webutvikling unngår vi å bruke "var"
  • let kan tilegnes en verdi som kan manipuleres og muteres fritt etter deklarering. Nyttig for tellere og verdier
  • const tilegnes en verdi som ikke kan overskrives. Den må tilegnes en verdi fra starten, men kan ikke få en ny verdi senere
var navnA;
let navnB;
const navnC = "Vi bruker likhetstegn (=) for å gi variabelen en verdi"

4. Grunnleggende datatyper

Det finnes totalt åtte datatyper i JavaScript; Streng, nummer, boolean, undefined, null, object, symbol og bigint. Vi avventer med de tre siste typene til et senere nivå.

4.1. String/Streng:

En Streng er en kombinasjon av tall og bokstaver og initialiseres med to anførselstegn. Det har ingenting å si om vi bruke enkel ' eller dobbel " anførselstegn, men vi kan ikke kombinere de for å starte og lukke samme streng.

const eksString = "Eksempel på en streng"

4.2. Integer/heltall og Floats/desimaltall:

I JavaScript er et heltall definert Et nummer uten fraksjoner, mens et desimaltall er definert som floats. For å tilegne en variabel verdien av et tall skriver vi det rett frem. Selv om Integer og Floats begge bruker tall, behandles de som to separate datatyper

const eksInteger = 1
const eksFloat = 6.12

4.4. Boolean

En Boolean er en datatype for om noe som har to tilstander; sant/aktivt eller usant/ikke aktivt. Dette er ofte brukt til å aktivere eller deaktivere funskjoner, enten direkte i koden eller via knapper

const eksBoolTrue = False
const eksBoolFalse = True

4.7. Undefined

Variabler som ikke har en definert verdi får verdien "undefined", fordi den ikke har fått en tilegnet verdi enda.

const eksUndefined = undefined

4.8. Null

Variabler som har verdien "null" har ingen objektiv verdi og kan heller ikke brukes som identifikasjon.

const eksNull = null

5. Operasjoner

Referere verdi

Verdien til en variabel kan hente og refere til andre variabler. I eksempelet under sier vi at variabel navnB har samme verdi som variabel navnA:

const navnA = 6
const navnB = navnA

Mattematiske operasjoner av tall

JavaScript kan brukes til gjøre nøyaktige utregninger med Integer/heltall og Floats/desimaltall opp til 15 sifrer. Vi skriver mattestykker rett frem eller i kombinasjon med nimeriske varibler, med addisjon+, substraksjon-, multiplikasjon* eller divisjon/. Eksempel:

const tallA = 6
const tallB = 3.5
document.getElementById("pluss").textContent = tallA + tallB;
document.getElementById("minus").textContent = tallA - tallB;
document.getElementById("gange").textContent = tallA * tallB;
document.getElementById("dele").textContent = tallA / tallB;

Sammenkoblingsoperatør / Sette sammen streng-verdier

Så lenge vi kun holder oss til addisjon, altså pluss+, kan vi legge sammen verdier og selvstendige streng-beskrivelser i form av et psudo-mattestykke. Bruker vi noe annet enn addisjon, får vi bare feilmeldingen "NaN", som betyr "Not a Number".

const tallA = 6
const navnA = "kant"
const seksKant = "Et heksagon er en " + tallA + "-" + navnA

Inkrement av tall

++ er en operatøren som kan brukes til å legge til en verdi eller inkrementere en eksisterende verdi. Her er et konkret eksempel hvor vi må bruke variabelen-typen let, fordi vi skal endre variabelens verdi etter at den er deklerert. Kombinert med en variabel, kan vi skrive opperatøren som i++, som er det samme som om vi skulle skrevet i = i + 1.

-- operatøren fungerer på akkurat samme måte. i--, er det samme som i = i - 1, og kan brukes til å trekke fra inkrement fra en verdi.

Kodeeksempel
let tallA = 10;
function leggTil() {
tallA++;
 document.getElementById("teller").textContent = tallA;
}
function trekkFra() {
tallA--;
 document.getElementById("teller").textContent = tallA;
}

Utregning med tilleggsargumet

I JavaScript er det ikke unormalt å definere verdien på en variabel, og like etterpå endre verdien eller legge til ytterligere verdi. Operatørenene +=, -=, *=, og /= er en snarvei for nettopp dette. tallA += 1 henter verdien til tallA og plusser på 1, som om vi hadde skrevet tallA = tallA + 1

Kodeeksempel
let tallA = 5;
tallA += 5;
tallA -= 5;
tallA *= 5;
tallA /= 5;

Tillegsargument + sammenkoblingsoperatør

Ettersom vi kan gjøre en sammenkoblingsoperasjon med +, kan vi kombinere det med tilleggsargumentet +=for en ryddigere kode.

Kodeeksempel
let navnB = "Etter første del, ";
navnB += "kommer del to.";

X. Avanserte datatyper

X.1. BigInt

BigInt er nemmeriske verdier som er for stor til å bli representert av vanlige nummer. BigInt verdier kan ikke utregnes som mattematisk med andre tall uten konvertering

const eksBigInt = BigInt("0x1fffffffffffff");

X.2. Objekt

Ett objekt er at datasett som består av flere nøkkelverdier. Objekter initialiseres med krøllparentes og nøklene separeres med komma

const eksObject = {name: "Bob", surname: "Kåre", age: 19,}

X.3. Symbol

Symboler er et ES6 datasett med en unik verdi som kan brukes som "nøkler" i objekter, slik at de ikke risikerer å overskrive eller kollidere med andre koder i et større datasett. Det kan også brukes for å definere skjulte egenskaper på objekter. Symbol() skrives med stor "S", og det er valgfritt om vi vil legge til en streng-beskrivelse i parentesen, eksempel Symbol("Nøkkel") Beskrivelsen har ingen innvirkning på symbolets unikhet, men kan refereres til i et objekt.

const eksSymbol = Symbol("nøkkel");
const eksObject = {name: "Bob", surname: "Kåre", [Nøkkel]: "Dette er en unik egenskap for objektet"}
